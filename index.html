Step 1 — Go to supabase.com, create a free account, then create a new project.
Step 2 — In your project go to Settings then API. Copy your Project URL and anon public key.
Step 3 — In your index.html file find these two lines at the top of the script and replace them:
const SUPABASE_URL = 'https://gjsdmdsxnacasjdbncndf.supabase.co'; const SUPABASE_KEY = 'sb_publishable_slk1_SXUzCAehzlBAkEGJQ_siqp6SJM'; Step 4 — In Supabase go to SQL Editor and run this to create the scores table:
create table naija_trivia_scores (player_name text primary key, streak int default 0, wins int default 0, played int default 0, best_streak int default 0, updated_at timestamptz default now()); Step 5 — In Table Editor, open naija_trivia_scores, click RLS and disable it so everyone can read and write.
